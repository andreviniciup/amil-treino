### **2. Execute os serviços (3 terminais):**

**Terminal 1 - Backend:**
```bash
cd backend
npm install
npx prisma migrate dev
npm run dev
```

**Terminal 2 - ML Service:**
```bash
cd ml-service
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

**Terminal 3 - Frontend:**
```bash
cd frontend
npm install
npm run dev
```

### **3. Teste tudo:**
```bash
.\TEST-ALL-ENDPOINTS.ps1
```
